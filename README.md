# TeachHub 

TeachHub is an open source database for K-12 educators to share and search for lesson plans. 


## npm Packages

* body-parser 
* dotenv 
* express 
* express-handlebars 
* mocha 
* mysql 
* mysql2 
* request 
* sequelize 
* supertest
